# 从 0 开始搭建我的个人博客

这是我的第一篇博客。

我希望这个博客以后主要用来记录：

- Java 后端学习笔记
- 面试题整理
- AI Agent 学习过程
- 项目复盘与技术总结

## 为什么要搭博客？

因为很多知识如果只是看过，很快就忘了；  
但如果自己整理、输出、沉淀，就会慢慢变成自己的东西。

## 第一阶段目标

先实现这几件事：

1. 本地可以访问
2. 能写 Markdown 文章
3. 能部署到 GitHub Pages
4. 后面再加域名、评论、统计
